# Kanban Mobile

Projeto Kanban migrado para React Native com Expo SDK 54, mantendo compatibilidade com Expo Go em dispositivo físico.

## Scripts

```sh
npm run start
npm run android
npm run ios
npm run web
npm run web:build
npm run typecheck
npm run lint
```

Use `npm run web` para abrir o board no navegador durante o desenvolvimento. Use `npm run web:build` para gerar uma versão web estática em `dist/`.

O board mantém criação, edição, remoção e arrastar para reordenar colunas e tarefas.

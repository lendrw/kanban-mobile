# Kanban Mobile

Projeto Kanban migrado para React Native com Expo SDK 54, mantendo compatibilidade com Expo Go em dispositivo físico.

## Scripts

```sh
npm run start
npm run android
npm run ios
npm run web
npm run typecheck
npm run lint
```

O board mantém criação, edição, remoção e arrastar para reordenar colunas e tarefas.
